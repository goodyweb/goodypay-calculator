# PayMongo pricing calculator for PHP

## Features
------------
```
Calculate the pricing of base on Paymongo specifically for **Card, E-Wallet and Online Banking**
```
## Installation
------------
Run

```
composer require goodypay/goodypay-calculator

composer install
```

## SRequirements
------------

```
Laravel version 9
PHP version 5.3.0

```
